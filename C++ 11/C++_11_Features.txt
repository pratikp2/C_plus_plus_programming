Features Added in C++ 11 :

-- Automatic type deduction
-- Attributes
-- lvaue and rvalue Reference
-- Constructor Deligation
-- Keyword final, override, delete, nullptr, default
-- move Semantics and move constructor
-- storage classes volatile and mutable
-- unique, shared and weak pointer (Smart Poointer) Improvement over auto pointer.
-- unordered set and multiset


 C++ has six kinds of special member functions:
•Default constructors
•Destructors
•Copy constructors
•Copy assignment operators
•Move constructors
•Move assignment operators
These special member functions are used to create, destroy, initialize, convert, and copy class objects.
